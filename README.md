# To Do
This is simple todo app written in sinatra, using Bootstrap css framework in front-end


## To install
- ``clone respository``
- ``Bundle install``
- ``rake db:create``
- ``rake: db:migrate``
- run ``rackup config.ru`` or ``bundle exec rackup -p1010 --host 0.0.0.0``
#  [Demo heroku website](https://todosinatra1.herokuapp.com).
